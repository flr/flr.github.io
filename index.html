FLR

The FLR library is a collection of tools for quantitative fisheries science, developed in the R language, that facilitates the construction of bio-economic simulation models of fisheries systems.

Features

  • Fully open source.
  • Extensible and adaptable to most fisheries.
  • Written in R and C++.
  • Heavily based on S4 classes.
  • Interacts nicely with many R packages.

Quick start

You can install the latest version of the FLR packages using our installation script

{% highlight r %}source("http://flr-project.org/R/instFLR.R"){% endhighlight %}

or, if you know you have all the necessary dependencies, directly from our repository

{% highlight r %}install.packages(repos="http://flr-project.org/R"){% endhighlight %}

and then maybe go through some tutorials.
